# Startup Cofounder Match

## Overview

Startup Cofounder Match helps entrepreneurs find potential cofounders based on shared skills and interests.

---

## Features

- Founder profile creation
- Skills management
- Interest management
- Cofounder matching
- Match score calculation

---

## Files

- `profile.py`
- `matcher.py`

---

## Usage

```bash
python profile.py
python matcher.py
```

---

## Example Output

```text
Founder Profile

Name: Sujal
Skills: Python, AI, Flask
Interests: EdTech, Startups


Match Result

{
    'match_score': 5,
    'common_skills': ['Python', 'AI'],
    'common_interests': ['Startups']
}
```
